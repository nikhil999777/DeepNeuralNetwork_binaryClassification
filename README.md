# Deep Neural Network for Binary Classification (From Scratch) | Python, NumPy

Built and implemented a multi-layer Deep Neural Network completely from scratch using Python and NumPy without relying on high-level deep learning frameworks.
Developed core neural network components including parameter initialization, forward propagation, backpropagation, gradient descent optimization, and cost computation.
Implemented ReLU and Sigmoid activation functions, cache management, and modular L-layer architecture for binary classification tasks.
Designed the complete training pipeline and validated model performance through iterative learning and parameter updates.
Demonstrated strong understanding of neural network mathematics, optimization, and machine learning fundamentals.
